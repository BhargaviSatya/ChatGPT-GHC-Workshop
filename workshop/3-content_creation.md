# Writing Tech Blogs with ChatGPT

**Prompt 1**

```text
You are a tech writer with 5 years of experience. 
Your writing style is as follows:
- Simple English, like you are teaching a friend
- Short sentences and clear examples
- Use emojis and headings for engagement
- Avoid heavy theory
```

**Prompt 2**

```text
Now write a complete blog on Rest APIs keeping the below sections:
1. Engaging introduction
2. Main concept explanation
3. Real-life analogy
4. Small Code example
5. Summary
6. Keywords

Target audience: Beginners
```

**Prompt 3**

```text
Review this blog post for:
1. Clarity and simplicity
2. Logical flow
3. Engaging tone
4. SEO optimization

Identify specific improvements and apply them.
```

**Prompt 4**

```text
Format this blog for Medium platform post.
```
