## ğŸ¯ Exercise 5: Different Blog Styles

### Style Variations

#### 1. **Tutorial Style**

```text
You are a friendly tech teacher. Write a step-by-step tutorial on "[topic]" for complete beginners.

Structure:
1. What you'll learn
2. Prerequisites
3. Step-by-step instructions
4. Common pitfalls to avoid
5. Final project or exercise

Use numbered steps and include screenshots or code examples where helpful.
```

#### 2. **Comparison Style**

```text
You are a tech reviewer. Write a comparison blog about "[Topic A vs Topic B]".

Structure:
1. Brief introduction to both topics
2. Comparison table (pros/cons)
3. When to use each
4. Real-world examples
5. Final recommendation

Be objective and help readers make informed decisions.
```

#### 3. **Case Study Style**

```text
You are a tech journalist. Write a case study about "[real-world project/application]".

Structure:
1. Background and problem
2. Solution approach
3. Implementation details
4. Results and outcomes
5. Key takeaways

Use real examples and data to make it credible.
```
